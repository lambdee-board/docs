# Docs

> Built with the [jekyll](https://jekyllrb.com/) gem
> and [just-the-docs](https://github.com/just-the-docs/just-the-docs) theme

## Install

```sh
$ bin/setup
```

## Run in development

Listens on `http://127.0.0.1:4000/`.

```sh
$ bin/dev
```

## Build for production

```sh
$ bin/build
```

## Deploy

Deploy to `https://lambdee.app`.

```sh
$ bin/deploy
```
